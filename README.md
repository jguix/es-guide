# EcmaScript guide from ES2015 to ES2019

## Build

Clone the repository and follow the instructions:

With `npm`;

```bash
$ npm install
$ npm start
```

With `yarn`:

```bash
$ yarn install
$ yarn start
```

As a result the browser will open a tab with the url http://localhost:3000 and you will see the page.
